# 🔒 CLOUDFLARE SSL - INSTALLAZIONE ISTEMDIATA

## 🎯 CONFIGURAZIONE RAPIDA (5 minuti)

### Passo 1: Crea Account Cloudflare
1. Vai su **https://dash.cloudflare.com/sign-up**
2. Registra account gratuito (email + password)

### Passo 2: Aggiungi Dominio
1. Accedi a **https://dash.cloudflare.com**
2. Clicca **"Add a Site"**
3. Inserisci: **agenziecase.com**
4. Seleziona piano **Free** → Continue

### Passo 3: Scan DNS Records
Cloudflare scannerà automaticamente i DNS esistenti.

### Passo 4: Aggiorna Nameserver OVH
1. Cloudflare ti darà 2 nameserver come:
   - **lara.ns.cloudflare.com**
   - **wesley.ns.cloudflare.com**

2. Vai su **OVH** → **Domini** → **agenziecase.com** → **DNS**
3. Cambia i nameserver con quelli di Cloudflare

### Passo 5: Attiva SSL
1. In Cloudflare vai su **SSL/TLS**
2. Imposta **"Flexible SSL"**
3. Attiva **"Always Use HTTPS"**

### Passo 6: Completa
Attendi **15-30 minuti** per la propagazione DNS.

## ✅ RISULTATO:
- 🔒 SSL gratuito
- 🌐 Nessun avviso browser
- 🚀 CDN inclusa
- 🛡️ Protezione DDoS
- ⚡ Performance migliori
