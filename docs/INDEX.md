## 🚀 Quick Start Guide

### Per Nuovi Utenti
1. Vai su [agenziecase.com](https://agenziecase.com)
2. Usa la ricerca per trovare immobili
3. Clicca ❤️ per salvare preferiti
4. Contatta l'agenzia per interessi

### Per Nuovi Partner
1. Registrati come agenzia sul portale
2. Attendi verifica (24h)
3. Pubblica il tuo primo immobile
4. Usa il CRM per gestire clienti

### Per Sviluppatori
1. Ottieni API token (contatta supporto)
2. Leggi [Documentazione API](API_DOCUMENTATION.md)
3. Usa sandbox per testing
4. Implementa integrazione
